# assets/

Drop `image_a.png` and `image_b.png` into this folder before launching the game to use your own visuals for **Subject A** (stable) and **Subject B** (anomalous).

Recommended:

- Square aspect ratio (512×512 or larger)
- PNG with transparency if you want clean compositing on CCTV feeds
- High contrast — the game applies heavy bloom, chromatic aberration, and scanline effects on top

If either file is missing, the game generates a placeholder at boot:

- Placeholder A: phosphor-green concentric-ring "eye" on a dark field
- Placeholder B: torn purple rectangles with a distorted off-center eye

Both placeholders are good enough to play through an entire loop — the images
exist to let you make the experience personal.
